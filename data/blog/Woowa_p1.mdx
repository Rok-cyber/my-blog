## 🧮 프로젝트 개요

문자열에 포함된 숫자를 구분자(기본/커스텀)를 기준으로 분리하고 합을 구하는 간단한 계산기를 구현했습니다.  
이번 미션은 단순한 계산기 구현이 아니라 **문자열 처리 로직 설계**, **예외 처리**, 그리고 **코드 스타일링**까지 신경써야 했던 프로젝트였습니다.

---

## 🧭 진행 목표

- GPT나 검색 없이 스스로 설계 및 구현
- 커스텀 구분자(`//;\n`) 처리 로직 완성
- 예외 처리 및 테스트 커버리지 확보
- Java Style Guide 준수 및 README 정리

---

## 🐞 가장 어려웠던 점

가장 시간을 많이 쏟았던 부분은 **커스텀 구분자와 줄바꿈(`\n`) 처리**였습니다.

- 한 줄 입력의 `\\n`을 실제 개행으로 치환하는 로직
- `//` 선언부와 second line 존재 여부 판단
- 잘못된 형식에서 즉시 예외를 던지는 `Validator` 설계

이 과정에서 수많은 디버깅을 반복했고, 그 과정에서 **입력 처리 설계의 중요성**을 크게 느꼈습니다.
